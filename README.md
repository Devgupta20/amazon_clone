# amazon_clone

A small static Amazon-like landing page built with plain HTML, CSS and JavaScript.

Features
- Hero image slider with auto-advance and manual next/previous controls.
- Search bar that redirects queries to Amazon (`searchAmazon`).
- Clickable sidebar menu that opens and closes (`openSidebar`, `closeSidebar`).
- Product-like feature boxes and a multi-column footer.

Files
- [index.html](index.html) — main page markup.
- [style.css](style.css) — styling for layout, slider, sidebar and footer.
- [script.js](script.js) — slider logic and UI handlers (`showSlide`, `nextSlide`, `prevSlide`, `searchAmazon`, `openSidebar`, `closeSidebar`).
- images/ — assets used by the page.

Run locally
1. Open [index.html](index.html) in your browser.  


Notes
- This is a static demo; external links point to Amazon.
- Edit styles in [style.css](style.css) and behavior in [script.js](script.js).
